\subsection{Detailed Description}
\mbox{\hyperlink{classNewConfigAppEditor}{New\+Config\+App\+Editor}} creates a pop-\/up editor component that allows the user to add a new application link to the list of favorite applications managed by \mbox{\hyperlink{classAppConfigFile}{App\+Config\+File}}. 

After entering data and pressing the confirm button, the new favorite application link is added to the menu and saved by \mbox{\hyperlink{classAppConfigFile}{App\+Config\+File}}.
