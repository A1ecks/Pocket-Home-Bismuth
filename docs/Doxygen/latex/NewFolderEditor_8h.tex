\subsection{Detailed Description}
\mbox{\hyperlink{classNewFolderEditor}{New\+Folder\+Editor}} creates a pop-\/up editor component that allows the user to add a new folder to the list of application folders managed by \mbox{\hyperlink{classAppConfigFile}{App\+Config\+File}}. 

After entering data and pressing the confirm button, the new folder is added to the menu and saved by \mbox{\hyperlink{classAppConfigFile}{App\+Config\+File}}.
