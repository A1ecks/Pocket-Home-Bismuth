void WifiStateManager::NetworkInterface::confirmWifiState ( )
protected

This method queries the wifi device to ensure that the current tracked wifi state actually matches the device state. If necessary, this will update the WifiStateManager's current state. The NetworkInterface should call this method whenever it has reason to believe the wifi state may be out of date.

◆ signalWifiConnecting()

void WifiStateManager::NetworkInterface::signalWifiConnecting ( )
protected

Whenever the wifi device detects a connection being established, the NetworkInterface should call this to notify its WifiStateManager.

This is intended to notify the state manager of unexpected connections, so it's not necessary to call this for connections opened by the state manager. Calls for known connection attempts will just be ignored.

Whenever the wifi device detects a connection being established, the NetworkInterface should call this to notify its WifiStateManager.
