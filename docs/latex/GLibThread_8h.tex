\subsection{Detailed Description}
\mbox{\hyperlink{classGLibThread}{G\+Lib\+Thread}} runs a G\+Lib event loop on a separate thread. 

On creation, this starts up a G\+Lib event thread to handle events associated with a G\+Main\+Context provided on creation. This G\+Main\+Context will be set as the thread default context. This context will be available through the get\+Context method, so that it can be used to add signal sources and signal handlers to this thread.

\mbox{\hyperlink{classGLibThread}{G\+Lib\+Thread}} also provides methods for synchronously or asynchronously executing code within the thread. These should be used to handle all interactions with G\+Lib object tied to the thread context, unless those objects are explicitly guaranteed to be thread-\/safe.
