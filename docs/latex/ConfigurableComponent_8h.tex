\subsection{Detailed Description}
\mbox{\hyperlink{classConfigurableComponent}{Configurable\+Component}} applies properties set in a \mbox{\hyperlink{classComponentConfigFile}{Component\+Config\+File}} to a component, updating those properties as necessary. 

This may set the component size and position relative to the window, change component asset files, and/or change image color values.
