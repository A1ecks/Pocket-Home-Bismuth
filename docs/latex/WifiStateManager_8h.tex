\subsection{Detailed Description}
\mbox{\hyperlink{classWifiStateManager}{Wifi\+State\+Manager}} tracks the current state of the Wifi network device, provides information about nearby access points and any current connections, notifies Listener objects whenever Wifi state changes, and provides an interface for requesting Wifi connections or other changes in Wifi state.

\mbox{\hyperlink{classWifiStateManager_1_1NetworkInterface}{Wifi\+State\+Manager\+::\+Network\+Interface}} is responsible for all direct communication with the Wifi device.
