\subsection{Detailed Description}
\mbox{\hyperlink{classConnectionSettingsComponent}{Connection\+Settings\+Component}} is a widget that provides a connection status icon, a toggle switch for enabling/disabling the connection, and a button that shows a connection status string. The button will also open a connection page when clicked while the switch is enabled.
