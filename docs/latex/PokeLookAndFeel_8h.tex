\subsection{Detailed Description}
sets the default appearance and behavior of all UI component objects. 

It sets UI values and defines custom draw routines for Juce sliders and buttons. It also passes default component colors from \mbox{\hyperlink{classComponentConfigFile}{Component\+Config\+File}} to all components.

Directly interacting with this class should not be necessary, except when the application calls Look\+And\+Feel\+::set\+Default\+Look\+And\+Feel() to set a \mbox{\hyperlink{classPokeLookAndFeel}{Poke\+Look\+And\+Feel}} object as the default.
