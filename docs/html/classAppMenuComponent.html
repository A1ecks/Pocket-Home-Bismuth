void AppMenuComponent::loadBaseFolder ( )

Load and display the base menu folder that contains favorite application shortcuts and all other folders. Any existing open folders will first be closed. If the base folder was open, its selected index will be saved and re-selected.

This method will run whenever this component is resized. Classes inheriting AppMenuComponent should override this instead of void resized(). This method will run before the menu folders are updated for the new bounds.
