String  psk = String() 
)
overrideprotectedvirtual

Triggers a wifi connection event. This will set a timer to simulate the connection's success or failure after a randomized delay of two to four seconds.

Parameters
toConnectIf this has the SSID "MyFi", a successful connection will be simulated. For all other access points, a failed connection will be simulated.
pskThe security key for WifiAccessPoint toConnect, or the empty string if the connection is not secured. This value will be noted in debug output, but otherwise is ignored.

Triggers a wifi connection event. This will set a timer to simulate the connection's success or failure after a randomized delay of two to four seconds.
